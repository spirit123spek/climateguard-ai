# ClimateGuard AI

AI-Based Climate Risk Assessment and Mitigation Platform Using Artificial Intelligence

## Key Features

* Real-time Weather Monitoring
* Live AQI Monitoring
* Climate Risk Prediction
* ESG Sustainability Scoring
* Historical Trend Analysis
* AI Climate Impact Simulator
* Climate Risk Alert Index
* Mitigation Recommendation Engine
* PDF Report Generation

## Important Requirement

All analytics, dashboards, predictions, and simulations must use REAL environmental data. No fake, static, hardcoded, or demo climate values should be used.

## Real Data Sources

* OpenWeather API
* AQICN API
* NASA EarthData
* India Open Government Data
* Historical Climate Datasets
